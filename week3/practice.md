```

```

#### ✅ 통합 서술 방식 (목표)

**주제별 그룹화:**

**주제 1: [예: 작업기억의 용량 제한]**
```
여러 연구들(Smith, 2020; Jones et al., 2021)은...
그러나 이들 연구는 공통적으로 ... 한계가 있다.
```

**주제 2: [예: 주의 자원 배분]**
```
최근 연구들(Brown, 2021; Davis & Lee, 2022)은...
특히 ... 조건에서 ... 패턴이 관찰되었다.
```

**주제 3: [연구 공백]**
```
그럼에도 불구하고, ... 여전히 해결되지 않았다.
본 연구는 이러한 공백을 ... 방법으로 채우고자 한다.
```
