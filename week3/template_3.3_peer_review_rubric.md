# Template 3.3: Peer Review Rubric (Gap Discovery)

**Week 3 - Research Gap Discovery & Validation**
**Activity**: Structured peer feedback on discovered and validated gaps (20 minutes)
**Canvas Size**: 3840×2160px (Figma)

---

## Purpose

Students provide structured peer review using a 4-dimension rubric:
1. **Gap Quality** (40%): Conceptual/Mechanistic/Translational vs Incremental
2. **Validation Thoroughness** (30%): Quality of 3-stage validation
3. **Gap Statement Clarity** (20%): Specificity and testability
4. **AI Tool Usage** (10%): Appropriate recipe selection and application

Output: Weighted score, balanced feedback (strengths + concerns), and Top 3 Priority Actions

## Left Panel: 평가 기준 참조

### 4-Dimension Rubric

```markdown
┌─────────────────────────────────────────────────────────────────┐
│ 🎯 PEER REVIEW RUBRIC (Gap Discovery)                           │
│ 4가지 차원으로 동료 평가                                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│ 1️⃣ GAP QUALITY (40% weight)                                    │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ ❓ Key Questions:                                                │
│ • Is this Conceptual/Mechanistic/Translational (not Incremental)?│
│ • Is it SURPRISING (counterintuitive predictions)?              │
│ • Is it CONSEQUENTIAL (changes theory or enables apps)?         │
│ • Is it TRACTABLE (testable with current methods)?              │
│                                                                  │
│ 📊 Scoring (1-5):                                                │
│ • 5/5: All 3 criteria met (Surprise + Consequential + Tractable)│
│   → Competitive for Nature/Science                              │
│ • 4/5: Strong on 2/3 criteria, adequate on 1                    │
│   → Competitive for specialized top journals (e.g., Nat Neuro)  │
│ • 3/5: Meets 1-2 criteria, but gaps in others                   │
│   → Needs refinement before submission                          │
│ • 2/5: Mostly incremental with minor novel elements             │
│   → Major revision needed                                       │
│ • 1/5: Purely incremental (population/context extension)        │
│   → Redirect to conceptual gap                                  │
│                                                                  │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ 2️⃣ VALIDATION THOROUGHNESS (30% weight)                        │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ ❓ Key Questions:                                                │
│ • All 3 stages completed? (Cross-disciplinary, Impact, Feasibility)│
│ • Evidence-based scoring? (not just intuition)                  │
│ • Realistic assessment? (not inflated scores)                   │
│                                                                  │
│ 📊 Scoring (1-5):                                                │
│ • 5/5: All 3 stages thorough + Evidence-based + Realistic       │
│ • 4/5: All 3 stages complete, minor gaps in evidence            │
│ • 3/5: All 3 stages attempted, but 1-2 need more depth          │
│ • 2/5: Only 1-2 stages completed                                │
│ • 1/5: Validation skipped or superficial                        │
│                                                                  │
│ 💡 Check: Which stage needs more work?                           │
│ □ Stage 1 (Novelty) - Need deeper cross-disciplinary search     │
│ □ Stage 2 (Impact) - Need specific stakeholders/actions         │
│ □ Stage 3 (Feasibility) - Need resource estimates/mitigation    │
│                                                                  │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ 3️⃣ GAP STATEMENT CLARITY (20% weight)                          │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ ❓ Key Questions:                                                │
│ • Is it SPECIFIC? (not vague "we will explore...")              │
│ • Is it TESTABLE? (clear predictions, not just descriptions)    │
│ • Is it CONCISE? (1-2 sentences for initial statement)          │
│                                                                  │
│ 📊 Scoring (1-5):                                                │
│ • 5/5: Specific + Testable + Concise (Nature/Science style)     │
│ • 4/5: Mostly clear, minor wording improvements needed          │
│ • 3/5: Core idea clear, but needs precision or structure        │
│ • 2/5: Vague or rambling (>3 sentences, unclear predictions)    │
│ • 1/5: Incomprehensible or missing                              │
│                                                                  │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ 4️⃣ AI TOOL USAGE (10% weight)                                  │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ ❓ Key Questions:                                                │
│ • Appropriate recipe selection? (matched to task)               │
│ • Effective prompting? (clear input, useful output)             │
│ • Critical evaluation? (didn't blindly accept AI)               │
│                                                                  │
│ 📊 Scoring (1-5):                                                │
│ • 5/5: Strategic use + Effective prompts + Critical evaluation  │
│ • 4/5: Good recipe choice, effective use                        │
│ • 3/5: Used AI, but could be more strategic                     │
│ • 2/5: Inappropriate recipes or poor prompts                    │
│ • 1/5: Didn't use AI or misused it                              │
│                                                                  │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                  │
│ 🎯 WEIGHTED OVERALL SCORE                                        │
│                                                                  │
│ Formula:                                                         │
│ (Gap Quality × 0.4) + (Validation × 0.3) + (Clarity × 0.2) +    │
│ (AI Usage × 0.1) = Overall /5.0                                  │
│                                                                  │
│ Interpretation:                                                  │
│ • 4.5-5.0: Excellent - Ready for top-tier submission            │
│ • 4.0-4.4: Very Good - Minor refinements needed                 │
│ • 3.5-3.9: Good - Solid foundation, some improvements needed    │
│ • 3.0-3.4: Adequate - Major improvements required               │
│ • <3.0: Needs Significant Work - Revisit gap discovery          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## Right Panel: Review Worksheets

### Round-Robin Pairing

Students review 2 peers in sequence:
- Student A reviews → Student B
- Student B reviews → Student C
- Student C reviews → Student D
- ...
- Last student reviews → Student A (closes loop)

Each student gives 1-2 reviews, receives 1-2 reviews.

## Facilitation Script

### Setup (2 min)

- Assign review pairs (round-robin)
- Orient: "Left panel = rubric, Right = your review zone"
- Emphasize: "Be honest AND constructive - this helps your peer"

### During Activity (20 min)

**Minutes 0-2: Calibration**
- Show 1 example gap (instructor-provided)
- Class scores together → Discuss differences → Calibrate

**Minutes 2-17: Individual Reviews**
- Students review assigned peer's work
- Monitor: Are scores evidence-based? Is feedback specific?
- Intervene if: Scores all 5/5 (too lenient) or all 1/5 (too harsh)

**Minutes 17-20: Share & Discuss**
- Students read reviews they received
- Optional: Respond to reviewer's suggestions
- Class discussion: Common themes? Surprises?

---

## Success Metrics

- All students complete review: >95%
- Feedback is specific (not vague): >80%
- Scores calibrated (not all 5/5 or 1/1): >90%
- Top 3 actions are actionable: >85%
