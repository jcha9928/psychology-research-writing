# Week 2-6 Figma 워크샵 전략

## 🎯 AI 시대의 협업 워크샵

Week 2-6는 AI 도구를 활용한 논문 작성 주차입니다. Figma를 활용하여:
- **AI 프롬프트 레시피 실시간 공유**
- **AI 출력 결과 비교 및 평가**
- **집단 지성으로 최적 프롬프트 발견**

```
│    - 새로운 이론적 통찰         │ ───────────────│        │

│    - 방법론적 개선              │ "이 분야에서    │        │
```

```
💬 교수자 중재:
"Red team의 지적이 타당하면 1점

 꼭 반영해야 합니다."
```

```
│  3. Discussion Implication 톤다운                            │
└─────────────────────────────────────────────────────────────┘
```

### Workshop: Round-Robin Review

```
🎭 Round-Robin 피어 리뷰

각 학생이 3명의 논문을 리뷰:

Round 1 (20분): Abstract + Introduction
↓
학생들이 Figma에서 다른 논문으로 이동
↓
Round 2 (20분): Methods + Results
↓
학생들이 다시 이동
↓
Round 3 (10분): Discussion + Overall
↓
원래 저자가 모든 피드백 통합

Figma에서 구현:
- 각 논문마다 별도 Frame
- 리뷰어들이 Sticky Note로 피드백
- 색상으로 리뷰어 구분 (학생A=노랑, B=파랑, C=초록)
- 저자가 피드백을 읽고 체크박스로 수용/거절 표시
```

## 💡 Pro Tips

### AI 도구 병행 전략

```
Figma + AI 도구 워크플로우:

```

### 익명성 vs 투명성

```
Week 2-4: 익명 가능
→ 학생들이 자유롭게 실험

Week 5-6: 실명 필수
→ 진지한 피어 리뷰를 위해
→ 책임감 있는 피드백
```

### 하이브리드 모드

```
일부 학생이 대면 참여 못할 때:

온라인 학생:
→ Figma에서 동일하게 참여
→ Zoom으로 음성 참여

대면 학생:
→ 교실 대형 스크린에 Figma 투사
→ 개인 노트북으로 편집

모든 작업이 Figma에 통합되어 누락 없음
```
